<objective>
Set AUTH_SECRET in the sandbox environment so Auth.js stops throwing MissingSecret errors.

Purpose: All three UAT gaps (Tests 2, 3, 6) share a single root cause — AUTH_SECRET is absent from the running process environment. Auth.js requires this secret to sign JWTs; without it, every auth-route request throws MissingSecret (500), which manifests as an upstream proxy error on login (Tests 2, 3) and a silently truthy `req.auth` error object that bypasses the middleware guard (Test 6). No logic changes are needed — only the env var must be present.

Output: `.env.local` (gitignored, read automatically by Next.js and Auth.js v5) and `.env.example` (committed, documents required vars).
</objective>

Depends on: None

</feature_dependencies>
