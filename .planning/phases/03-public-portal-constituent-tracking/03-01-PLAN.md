  ```typescript

  delete (L.Icon.Default.prototype as any)._getIconUrl;

  ```
  Show a dropdown of results; on select, fly map to that location, place a marker, call `onLocationChange`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→API | Untrusted browser request to GET /api/categories |
| db→render | Category data from Postgres reflected into HTML select options |
| client→Nominatim | User-typed address string sent to third-party geocoder |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01-01 | Denial of service | GET /api/categories | mitigate | `Cache-Control: public, max-age=60` in `app/api/categories/route.ts` response header limits DB reads to at most once/minute per CDN node; Prisma query uses indexed `active=true` filter |
| T-03-01-02 | Information disclosure | GET /api/categories | mitigate | Route returns only non-sensitive category metadata (no PII, no staff data); `app/api/categories/route.ts` `findMany` select projection omits internal fields |
| T-03-01-03 | Tampering | db→render (category options in select) | mitigate | React escapes all values rendered into `<option>` elements; no `dangerouslySetInnerHTML` in `app/(public)/page.tsx` category picker |
| T-03-01-04 | Information disclosure | client→Nominatim (address search) | accept | Nominatim is a public OSM service; search queries are not sensitive (addresses). Residual risk: Nominatim rate limits could reveal to OSM that the city runs this portal. Accepted — no API key or PII sent. |
| T-03-01-05 | Denial of service | client→Nominatim (address search) | mitigate | Nominatim fetches in `components/maps/ReportingMap.tsx` are debounced (300ms) on keystroke and rate-limited client-side; `User-Agent: uReport-NG` header set per OSM usage policy |
</threat_model>
